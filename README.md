# Tailwind Preset
Tailwind preset for Emry UI

### Installation

```
yarn add @emryui/tailwind-preset
```

`tailwind.config.js`:
```
module.exports = {
  presets: [
    require("@emryui/tailwind-preset")
  ],
  // Everything else
}
```

### What's included
- [x] Colors (primary, gray, success, error, warning, +24 others)
- [x] Typography
- [x] Gradients (primary & gray)
- [x] Container (sm, md, lg, xl)
